线性代数笔记整理

线性代数Re

行列式

逆序数

  • 自然排列：就是123456789这样顺序排列
  • example : $\tau$(5,4,3,2,1)=4+3+2+1+0=10
  • 逆序数的奇偶决定了是奇数排列还是偶数排列

    n阶行列式

    n阶行列式有n!项因子

    特殊行列式

  • 上/下三角行列式：等于对角线的乘积

    行列式性质

  • $|A^T|$ =$| A |$
  • 交换行列式两行（列），行列式变号
  • 行列式有两行（列）成比例包括相同，则行列式为0
  • 行列式的一行（列）有公因子k，k可以提到行列式前。
  • 线性拆分定理
  • 行列式某行（列）的倍数加到另一行（列），行列式值不变

    行列式的计算

    思想：利用行列式的性质，简化行列式运算，将行列式归结为三角行列式的计算

行列式按行按列展开

定理：行列式等于它的任一行（列）的各元素与其对应的代数余子式乘积之和

范德蒙行列式

V(x1,x2,…，xn)=$\prod_{1\leq j \leq i \leq n}{x_i-x_j}$

矩阵运算

  • 矩阵的转置

    • (A+B)T
    • $(A\cdot B)^T=A^T \cdot B^T$
    • $(kA)^T=k\cdot T$

    若矩阵$A^T=A$那么矩阵A被称为对称矩阵

计算机控制系统笔记整理

计算机控制系统

线性离散系统的数学描述和分析方法

计算机控制系统本质上是一个线性离散系统。

信号变换理论

  • 连续控制系统：给定量、反馈量和被控量都是连续型的时间函数（模拟信号）。
  • 离散控制系统：给定量、反馈量和被控量是在时间上离散的数字信号。

采样过程

  • 以一定时间间隔对连续信号进行采样，使连续信号转换为时间上离散、幅值上连续的脉冲序列的过程称之为采样过程。

  • 在计算机控制系统中，采样信号f*(t)使一数字序列，可分解为一系列单脉冲之和。

量化过程

  • 所谓量化，就是采用一组数码来逼近离散模拟信号的幅值，将其转化为数字信号的过程。

    • 量化单位：假设fmax和fmin分别为信号的最大值和最小值，则量化单位为：$q = {f_{max} -f_{min}\over 2^n}$ 其中n为二进制数字长。 通俗点说就是量化的位数，比如8位16位之类的

    • 对采样信号进行编码时，只能用q的整数倍来表示，这个量化过程由A-D转换器来完成，有两种量化方法：

      • 只射不入只舍不入
      • 有射有入有舍有入

    • 有了舍和入，就会产生误差：

      $e(t) = f(t) - f^*(t)$

    • 采样过程中如果频率足够高，并且选择足够大的字长，就可以使量化的误差足够小，同时使量化噪声的振幅小、频率高、易被被测对象抑制。

香农（Shannon）采样定理:

​ $\omega_s \geq 2\omega_{max}$

  • 保证采样信号沿频率轴方向
  • 以便采样信号额能够反应该信号的变化规律
  • 采样频率至少应该是输入信号频谱的最高频率$\omega_{max}$的两倍

现代控制理论笔记整理

控制系统的空间表达

状态变量及状态空间表达式的结构图

状态变量：

定义： 状态变量是既足以完全确定系统运动状态而个数又是最小的一组变量，当其在 t = t0时刻的值已知时，则在给定t≥t0时刻的输入作用下，便能完全确定系统在任何t≥t0时刻的行为。

状态矢量：

如果n个状态变量用x1,x2,……,xn表示，并把这些状态变量看作是矢量x(t)的分量，则x(t)就称为状态矢量，记作：

xT=[x1(t),x2(t),…,x(t)]

输出方程：

在指定系统输出的情况下，该输出与状态变量间的函数关系式，称为系统的输出方程。

yT=[y1(t),y2(t),…,ym(t)]

状态空间表达式：

  • 状态方程和输出方程
  • 非唯一性：状态变量选择不同则状态方程不同
  • SISO定常系统状态空间表达式
    • $\dot{x}$ = Ax+bu
    • y=cx
  • MIMO定常系统状态空间表达式
    • $\dot{x}$=Ax+Bu
    • y=Cx+Du

状态变量及状态空间表达式的模拟结构图

约旦变换

定义：约旦变换是将$\dot{x}$ = Ax + Bu，y=Cx变换为$\dot{z}$ =$J \cdot z$ + T -1$ \cdot{z}\cdot{u}$ ,y=C$\cdot T\cdot z$

步骤：

  • 根据系统矩阵A，求A的特征值，可以得到系统约旦标准型矩阵J
    • 有重根时的情况
    • 无重根的情况
  • 求变换的控制矩阵T-1B和输出矩阵CT，则必须求出变换矩阵T。
    • A阵位任意形式
      • 特征值有重根时
      • 特征值无重根时
    • A阵为标准型
      • 特征值有重根时
      • 特征值无重根时
      • 特征值有共轭复根时

传递函数

步骤：

  • 由状态空间表达式$\dot{x}$=$A\cdot x$+$b\cdot u$，y=$c\cdot x$列出系统矩阵A和系数矩阵b

  • 根据Wux=($s\cdot I$-A)-1$\cdot b$求出状态传递函数

    • X(s) = (s$\cdot I$-A)-1B$\cdot U(s)$

  • 根据Wuy=c(s$\cdot I$-A)-1$\cdot b$求出输入输出传递函数

    • Y(s) = [C(s$\cdot I$-A)-1$\cdot B$+D]$\cdot U(s)$

【项目】A World，AI物联网管理终端

A World，AI物联网管理终端

前言

  • 初衷：做这么一个基于AI的物联网管理终端的出发点其实只是自己想自娱自乐一下，同时也做一些能够送的出手的东西，也是为了不咕咕某人。

  • 直到看到了OPPO算法工程师稚晖的AEYE才最终确定了构思。

  • 平台：嘉楠勘智K210（RSIC-Ⅴ架构），华为NB-IOT（4G物联网）

初探

这就要从单片机课的课程设计说起了，单片机课程设计，我使用了Sipeed的K210做了人脸识别的一个玩意儿，在小小的单片机上运行神经网络是一件几乎闻所未闻的事情。以往对神经网络的印象都是运算量大，没好电脑带不动之类的。实际上这是错误的，模型的训练的确是一件很费算力的事情，但是运行已经训练好的模型，并不是件费力的事情。普通的单片机，彩摄都很难采集，比如51架构的单片机。而K210是第一个双核64位RSIC-Ⅴ的芯片，运行主频可以达到600Mhz。你可能会说，600mhz，NXP的i.mx rt系列的也能达到这个速度。最大的亮点来了，这是K210的架构图：

  • 你可以发现他的配置令人兴奋：

    • K210 包含 RISC-V 64 位双核 CPU，
    • 每个核心内置独立 FPU.
    • K210 的核心功能是机器视觉与听觉，其包含用于计算卷积人工神经网络的 KPU 与用于处理麦克风阵列输入的 APU.
    • 同时 K210 具备快速傅里叶 变换加速器，可以进行高性能复数 FFT 计算。因此对于大多数机器学习算法，K210 具备高性能处理能力。
    • K210 内嵌 AES 与 SHA256 算法加速器，为用户提供基本安全功能。
    • K210 拥有高性能、低功耗的 SRAM，以及功能强大的 DMA，在数据吞吐能力方面性能优异。
    • K210 具备丰富的外设单元，分别是：DVP、JTAG、OTP、FPIOA、GPIO、UART、SPI、RTC、I²S、I²C、 WDT、Timer 与 PWM，可满足海量应用场景。

Begin with End

从一次结束开始

  • 2020年3月15日

  • 晴

    • 我们趴在湖畔的栏杆上，我的左耳，你的右耳，一首陈粒未发布的单曲（你说富贵不求），一遍又一遍，将思绪缠绕牵扯到了高中时代。什么都能聊起，什么都能回忆，只有水里的鱼儿知道我们讲了什么……高中三年，大学亦是三年。我不会像曾经的我，你也不再是当初那个你。此时我知道，我们的青春，亦如入水的夕阳，快结束了。第一次的约会，诉尽衷肠，就这么多了。我知道，曾经的美好，已经散去了，高中三年的追逐，大学三年最艰苦岁月里的回忆，就像这水面的涟漪，一层层激荡开来，最终归于平静。曾经躲在宿舍被子里，借着好哥们的手机和你聊到天亮的日子，曾经游历欧洲给你寄的明信片，曾经争风吃过的醋，曾经凑钱送你的星空，曾经的疯狂、幼稚、冲动……所有的一切都已然成了过眼的云烟。眼前的，你夕阳下渐变的发色，以及浅蓝色的虹膜，还是那么美，你指指远处的鸭子，我指指跟前带点蓝色的鱼儿，多么希望时间是能够定格在那一刻，在那一刻，仿佛还是趴在教学楼栏杆上对望的你我，仿佛还是那些时日憧憬的心情。

    • 但我知道的是，时代变了，大人。没人能够像扎西拉姆·多多诗里说的那样留在岁月的岸上。曾经的我把说不完的话写在笔记上，写了这么三年，如今再没翻开那本笔记的必要了。这次见面是时候告别了，告别曾经的那段青春，告别那个还停留在日记本中的你。大学三年时间，几次简短的寒暄，几句草草唱完的生日歌，不知是我的倔强还是愧疚。倔强的是还挂着你这件事，愧疚的是我离你远去的懦弱。我麻痹自己在学生工作里，麻痹自己在钢琴丝弦以及热爱的事业上。下次也许还会见面，而再见面就是另一种心情了，故事也是另一段故事了。至此，心情却难以平复，悲伤席卷我，撕咬我每一寸五脏六腑。
